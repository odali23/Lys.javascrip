¿Qué son las variables?

Las variables son espacios en la memoria donde puedes guardar información que puede cambiar durante la ejecución del programa. En JavaScript, puedes crear variables usando let, const o var. 

// Declaración de variables
let edad = 25;
const nombre = "Lys";
var ciudad = "Madrid";

console.log(nombre + " tiene " + edad + " años y vive en " + ciudad);

¿Qué son los objetos?

Los objetos en JavaScript permiten agrupar datos y funciones relacionadas en una sola entidad. Se crean usando llaves {} y pueden tener propiedades y métodos. 

// Crear un objeto
let persona = {
    nombre: "Lys",
    edad: 22,
    saludar: function() {
        return "Hola, soy " + this.nombre;
    }
};

console.log(persona.nombre);
console.log(persona.saludar());

¿Qué son los bucles?

Los bucles permiten ejecutar un bloque de código varias veces. Los tipos más comunes en JavaScript son for, while y do...while. 


// Bucle for
for(let i = 1; i <= 5; i++) {
    console.log("Repetición número: " + i);
}

// Bucle while
let contador = 1;
while(contador <= 3) {
    console.log("Contador: " + contador);
    contador++;
}

// Bucle do...while
let x = 1;
do {
    console.log("Valor de x: " + x);
    x++;
} while(x <= 2);
            
            
            

        

        

        
        

            
            
¿Qué son las funciones?

        

        

            

                Las funciones son bloques de código que puedes reutilizar. Se definen con la palabra function y pueden recibir parámetros.
            

            
// Declarar una función
function saludar(nombre) {
    return "Hola, " + nombre + "!";
}

console.log(saludar("Lys"));
            

            
            

        

        

        
        

            
            
¿Qué son los arreglos?

        

        

            

                Los arreglos son listas que almacenan varios valores. Puedes acceder a cada elemento usando su índice.
            

            
// Crear un arreglo
let frutas = ["Manzana", "Banana", "Naranja"];
frutas.push("Uva");
console.log(frutas);
            

            
            

        

        

        
        

            
            
¿Qué son las condicionales?

        

        

            

                Las condicionales permiten ejecutar diferentes bloques de código según si una condición es verdadera o falsa. Se usan if, else if y else.
            

            
// Ejemplo de condicional
let edad = 18;
if(edad >= 18) {
    console.log("Eres mayor de edad");
} else {
    console.log("Eres menor de edad");
}
