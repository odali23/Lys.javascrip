Lenguajes de Programación

HTML

HTML (HyperText Markup Language) es el lenguaje principal para crear páginas web.

<h1>¡Hola Mundo!</h1>

CSS

CSS (Cascading Style Sheets) se utiliza para dar estilo a las páginas web.

h1 {
    color: blue;
    font-size: 2em;
}

JavaScript

JavaScript es un lenguaje de programación que permite agregar interactividad a las páginas web.

console.log('¡Hola Mundo!');

Java

Java es un lenguaje de programación orientado a objetos, ampliamente usado en aplicaciones empresariales y móviles.

public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("¡Hola Mundo!");
    }
}
